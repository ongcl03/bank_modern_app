The application was created as a learning exercise by following the tutorial on YouTube: [Build and Deploy a React Website with Tailwind CSS - React JS Project](https://www.youtube.com/watch?v=_oO4Qi5aVZs).

## Getting Started

To run the project locally, follow these steps:

1. Clone the repository: `git clone https://github.com/ongcl03/bank_modern_app.git`
2. Navigate to the project directory: `cd bank_modern_app`
3. Install dependencies: `npm install`
4. Start the development server: `npm run dev`
5. Open your browser and visit `http://localhost:5173` to view the application.
